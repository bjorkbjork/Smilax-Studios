ANZAC COVE - GALLIPOLI

ABOUT THE GAME

This game is a low-poly, goofy shooter set in the invasion of Gallipoli, WW1. Despite this historic setting, the game takes place on an entrily fictitious set of islands. Currently on version Pre-Alpha 2.1, expected to be updated to 3.0 within the next week. In 3.0 you can expect:
• A shooting mechanic,
• A hittable model, with:
    • Working hitboxes
    • Different damage for certain hitboxes
 And much, much more planned for the future.
