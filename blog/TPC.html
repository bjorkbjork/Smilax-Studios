DevBlog 1.0

26/09/18



We are currently waiting on all of the Pixel Art assets to be finished by Sam. All of the base code has been added, and just needs some tweaking.

If you have any suggestions for what you would like to see in this game -- it's currently just a clicker -- then email us your suggestions!

Pre-Alpha 1.0 is expected to have:

• All the art assets completed.
• A working settings menu
• A working 'E's earnt per second' feature.
